# qa_portfolio_YP

Учебный проект по тестированию API.

В рамках проекта я проверял API-методы на тестовом стенде, оформлял тестовую документацию, фиксировал дефекты и собирал артефакты, подтверждающие результаты тестирования.

## О проекте

Проект посвящён ручному тестированию API.  
Для проверки использовались Postman и документация Apidoc.  
В ходе работы я проверял корректность ответов, статусы, структуру JSON, обязательные поля, а также положительные и отрицательные сценарии.
